# Library catalog

The Docs tab shows a detail page for each library listed below. Tap a library name in the table of contents to open its `.md`.

## Images & Graphics
- **Pillow** — PNG/JPEG/drawing/text (full iOS build)
- **cairo (pycairo)** — vector 2D graphics, SVG/PDF/PNG output

## Numerical Computing
- **numpy** — arrays & linear algebra (OpenBLAS)
- **scipy** — stats, optimize, signal, interpolate
- **sympy** — symbolic math
- **networkx** — graph theory & algorithms

## Machine Learning
- **scikit-learn** — classical ML
- **torch** — PyTorch (native iOS build)
- **transformers** — Hugging Face models
- **huggingface_hub** — model hub client

## Plotting
- **matplotlib** — pyplot API (plotly-backed)
- **plotly** — interactive charts

## Animation
- **manim** — math animation → MP4

## Media
- **av (PyAV)** — FFmpeg bindings (video + audio)

## System
- **psutil** — CPU / memory / process info

## Networking
- **requests** — HTTP client
- **beautifulsoup4** — HTML/XML parser

## Terminal
- **rich** — colored output, tables, progress (install via pip)
- **pygments** — syntax highlighting
- **tqdm** — progress bars
